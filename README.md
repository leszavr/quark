# Quark

Quark - это система, спроектированная и реализованная с участием ИИ.

## Структура проекта

- **auth-service** - Сервис аутентификации
- **user-service** - Сервис управления пользователями
- **blog-service** - Сервис блогов
- **messaging-service** - Сервис обмена сообщениями
- **media-service** - Сервис управления медиа
- **ai-orchestrator** - Оркестратор ИИ

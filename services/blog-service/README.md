# Blog Service

Ядро блоговой платформы Quark.

## Функции
- Мультисайтовые блоги
- Блоковый контент (WYSIWYG)
- Черновики и публикация
- SEO через SSR
- Интеграция с media-service и ai-orchestrator

## API
Swagger: `/docs`

## Мультисайтовость
Поддержка:
- /user/alex
- alex.blog.quark.ai
- myblog.com (custom domain)

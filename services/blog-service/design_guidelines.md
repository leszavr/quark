# Дизайн-гайдлайны для BlogChat

## Выбранный подход дизайна
**Система дизайна:** Material Design адаптированная для чат-приложения
**Обоснование:** Приложение сочетает социальные функции (блог) и утилитарные (мессенджер), требует читаемости контента и интуитивной навигации.

## Основные принципы дизайна
- Чистота и минимализм для фокуса на контенте
- Плавные переходы между состояниями интерфейса
- Адаптивность для всех устройств
- Доступность и читаемость текста

## Цветовая палитра

### Шесть цветовых тем:
1. **Classic:** 220 15% 25% (основной), нейтральные серые
2. **Blue:** 215 85% 55% (основной), холодные оттенки  
3. **Purple:** 270 75% 60% (основной), фиолетовые акценты
4. **Green:** 145 70% 45% (основной), природные тона
5. **Orange:** 25 85% 55% (основной), теплые оттенки
6. **Indigo:** 235 80% 50% (основной), глубокие синие

### Общие цвета для всех тем:
- Фон светлый: 0 0% 98%
- Фон темный: 220 15% 8%
- Текст: 220 15% 15% / 0 0% 90%
- Границы: соответствующий основной цвет с 20% прозрачностью

## Типографика
**Основной шрифт:** Inter (Google Fonts)
**Заголовки:** 24px-32px, font-weight: 600
**Основной текст:** 14px-16px, font-weight: 400
**Мелкий текст:** 12px-14px, font-weight: 400

## Система отступов
**Базовые единицы:** 2, 4, 6, 8 (Tailwind)
- Маленькие элементы: p-2, m-2
- Средние блоки: p-4, m-4, gap-4
- Большие секции: p-6, m-6, p-8, m-8

## Компоненты

### Навигация
- Фиксированный хедер с логотипом, переключателем тем, статусом пользователя
- Минималистичный дизайн с четкими границами

### Блог-лента
- Карточки постов с rounded-lg границами
- Метаданные автора с аватарами (32x32px)
- Кнопки лайков и комментариев с иконками Lucide

### Мессенджер
- Сворачиваемая правая панель (300px в развернутом виде)
- Список сообщений с временными метками
- Поле ввода с кнопкой отправки

### Модальные окна
- Полупрозрачный фон (bg-black/50)
- Центрированное содержимое с rounded-xl
- Плавная анимация появления/исчезновения

### Кнопки и формы
- Radix UI компоненты с кастомной стилизацией
- Состояния hover с изменением яркости на 10%
- Фокус с outline в цвете темы

## Анимации
**Минимальные переходы:**
- transition-all duration-200 для hover эффектов
- transform scale для кнопок при нажатии
- opacity transitions для модальных окон

## Адаптивность
- **Desktop:** полный интерфейс с боковым мессенджером
- **Tablet:** сворачиваемый мессенджер
- **Mobile:** стековая компоновка, полноэкранный чат

## Изображения
- Аватары пользователей: круглые, 32px и 48px
- Логотип: простая иконка чата в хедере и футере
- Без героических изображений - фокус на контенте

Дизайн подчеркивает функциональность и удобство использования, обеспечивая чистый современный интерфейс для эффективного общения и чтения контента.
